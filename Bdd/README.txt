Marche avec une bdd de nom : bdd_membre

Les tables sont normalement créées dans le connex.inc.php si elles n'existent pas, donc la bdd n'est pas forcement necessaire c'est un plus.
A noter qu'on ne peut pas importer une bdd si les tables sont deja créées mais que l'on peut se connecter sur une base deja créée, il faut alors référer le bon nom de base.
